All about Fizz Buzz

Fizz Buzz is a simple and popular coding challenge often used to teach basic programming concepts like loops, conditionals, and modulo operations. The task is to write a program that iterates through a sequence of numbers, typically from 1 to 100, and prints a specific output based on certain conditions:

  1. If the number is a multiple of 3. Say Fizz!
  2. If the number is a multiple of 5. Say Buzz!
  3. If the number is a multiple of 3 and 5. Say FizzBuzz!
  4. Otherwise, just say the number.

Lets ask chatGPT to write us a FizzBuzz program in JavaScript and see what we get.

for (let i = 1; i <= 100; i++) {
if (i % 3 === 0 && i % 5 === 0) { console.log("FizzBuzz");
} else if (i % 3 === 0) { console.log("Fizz");
} else if (i % 5 === 0) { console.log("Buzz");
} else { console.log(i); }
}
